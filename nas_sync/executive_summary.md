## 🎯 核心功能

### 工作流程很简单

**第 1 步**：员工在飞书群组分享文档
```
员工：这是我们的 Q2 运营方案
链接：https://xxx.feishu.cn/docx/...
（无需任何额外操作）
```

**第 2 步**：系统自动处理（无需人工）
```
✓ 自动检测新分享
✓ 自动下载文档
✓ 自动按部门分类
✓ 自动建立索引
```

**第 3 步**：高效获取信息
```
高层或员工: 询问 AI 机器人 "Q2 运营方案是什么？"
机器人: "根据知识库，Q2 运营方案包括..."
        （附加相关数据和方案详情）
```

### 支持的文档类型

- ✅ **Word 文档** → 自动转换为可搜索文本
- ✅ **PDF 文件** → 自动转换为可搜索文本
- ✅ **表格/数据** → 自动转换为 Excel
- ✅ **所有其他格式** → 自动归档管理

## ❓ 常见问题

**Q: 员工需要改变工作习惯吗？**
A: 不需要。员工照常在飞书分享文档，系统自动处理。无额外操作。

**Q: 现有文档如何处理？**
A: 可以上传历史文档到群组，系统一次性处理。或者仅管理新文档。

**Q: 如果员工离职，文档怎么办？**
A: 系统自动保存，不受员工变化影响。这是主要收益之一。

**Q: 信息安全有保障吗？**
A: 是的。系统基于公司内部 NAS，数据不出公司网络。权限管理严格。

**Q: AI 机器人怎样保证准确性？**
A: 机器人基于真实文档回答，减少编造。同时标注来源，便于验证。

**Q: 能不能按需关闭？**
A: 可以。系统可随时停用，不影响其他业务。
