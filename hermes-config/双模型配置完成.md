## ⚠️ 注意事项

### MiniMax 注意事项：
1. **服务区域**: 使用中国区服务器 (`minimax.a7m.com.cn`)
2. **网络要求**: 确保能访问 `https://minimax.a7m.com.cn`
3. **速率限制**: 遵循 MiniMax API 限制
4. **上下文窗口**: 16K tokens（相对较小）

### GPT-5.4 注意事项：
1. **Plus 账户**: 需要 ChatGPT Plus 订阅
2. **OAuth 认证**: 使用 OpenAI Codex OAuth，无需 API Key
3. **自动刷新**: 凭证会自动刷新
4. **速率限制**: 遵循 OpenAI 限制

### 智能路由：
1. **自动切换**: 系统会根据任务复杂度自动判断
2. **手动覆盖**: 可以随时手动切换模型
3. **透明度**: 每次响应都会显示使用的模型
